Sketch Filter

A Photo to Pencil Drawing Converter

Category: Digital Image Processing

Project Type: Group

Date: May 2018 - June 2018

Role: MATLAB Programming, Essay, Poster Design


Abstract
Digital image processing technology has been applied to many aspects in our daily life nowadays. We wanted to develop a filter effect that can turn a photograph into a pencil-drawing-like image.

The topic of this project is in the intersection of computer technology and visual design. It serves as a funny tool that is able to bring entertainment to people’s life.

This approach can be used as a filter function of today’s photo editing softwares.The algorithms we used is more efficient than former approaches and the performance is also better.
